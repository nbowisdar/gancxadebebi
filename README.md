You need python version at least 3.11


## Prerequisites
1) Create new venv if needed
```bash
python -m venv venv
```
3) Activate venv
```bash
source venv/bin/activate
```

3) Install libraries
```bash
pip install -r requirements.txt
```

## Running app
```bash
python -m app
```


## Configuration 

To configure script open config.py and change what you need.
